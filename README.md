# visionai-data

Данные, стиль-гайд и задачи для Claude-автоматизации VisionAI.

## Структура

```
tasks/
  tg-morning-news/      — Утренняя AI-новость в Telegram (14:00 MSK)
  tg-feature-prompt/    — Фича/промпт в Telegram (18:00 MSK)
  tg-evening-guide/     — Гайд/кейс в Telegram (21:00 MSK)
  avito-kpi-report/     — KPI-отчёт Авито в Telegram (14:00, 18:00, 22:00 Самара)
  avito-ads-check/      — Мониторинг скрытых объявлений Авито (каждые 2ч, 11–23 Самара)

channel_style_guide.md  — Стиль-гайд для Telegram-канала
posted_history.json     — История опубликованных постов
sources.json            — Источники для новостей/контента
```

## Задачи (Scheduled Tasks)

| Задача | Расписание | Что делает |
|--------|-----------|------------|
| `tg-morning-news` | 14:00 MSK ежедневно | Ищет свежую AI-новость, публикует в Telegram-канал. Ведёт историю в `posted_history.json` |
| `tg-feature-prompt` | 18:00 MSK ежедневно | Пост про фичу или промпт Claude/AI. История в `posted_history.json` |
| `tg-evening-guide` | 21:00 MSK ежедневно | Гайд или кейс использования AI. История в `posted_history.json` |
| `avito-kpi-report` | 14:00, 18:00, 22:00 **Самара** | Собирает KPI с Авито (просмотры, контакты, CR, расходы), считает норму 25 контактов/4ч, шлёт отчёт @CleverlyUno + @IS_eg в Telegram |
| `avito-ads-check` | 11:00, 13:00, 15:00, 17:00, 19:00, 21:00, 23:00 **Самара** | Проверяет скрытые объявления на Авито. Если есть скрытые — шлёт алерт @CleverlyUno + @IS_eg. Если все активны — молча завершает |

## Авито — логика смены

Смена начинается в **11:00 Самара**. Отчёты по KPI идут через каждые 4 часа от старта:
- 14:00 — 3ч в смене (ожидаемые контакты: ~19)
- 18:00 — 7ч в смене (ожидаемые контакты: ~44)
- 22:00 — 11ч в смене (ожидаемые контакты: ~69)

KPI пороги: CR > 15% ✅ | цена контакта < 150₽ ✅ | расход < 12 000₽/день ✅

## Файлы данных

- **`channel_style_guide.md`** — голос канала, антипаттерны, нативные вставки
- **`posted_history.json`** — дедупликация: что уже постили (URL/заголовок)
- **`sources.json`** — RSS/сайты откуда брать AI-новости
